Intro to Git & Github
Command Description 
git init
 Initializes our local repoository (repo) [ONLY RUN ONCE, when the repo is first made] 
git status
 displays status of files being tracked 
git add
 stages any changed files that you tell it to 
git rm
 UNstages any changed files that you tell it to 
git commit -m "your message"
 clears the staging, saves the changes on your LOCAL repo, and provides a "Why" 
git push
 upload your commits to a remote repository 
git remote add name "link"
 add a link to a remote repository that we can share information with
